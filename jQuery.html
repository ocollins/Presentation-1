jQuery Overview

  • jQuery is a fast, small, and feature-rich JavaScript library.
  • It was designed in 2006 to simplify client-side scripting of HTML.
  • Is the most popular JavaScript library in use today, with installation on 65% of the top 10 million highest-trafficked sites on the Web
  • Makes it easy to manipulate HTML document, handling events, and working with animation.
  • It provides an easy-to-use API that works across a multitude of browsers including older ones.
  • There are two ways to include jQuery into your page:
    You can download it from jquery.com and include the script tag:
    < script src="jquery-3.1.1.min.js"></script>
    or use a Google or Microsoft-hosted version.
    you have to include one of these into your HTML:
    <script type = "text/javascript" src = "https://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js"></script>
    <script type = "text/javascript" src = "https://ajax.aspnetcdn.com/ajax/jQuery/jquery-3.1.1.min.js"></script>
  • jQuery syntax is tailor-made for selecting HTML elements and performing action on the elements.
    The syntax is: $("selector").action()
  • jQuery commands have to be enclosed into <script> element if you are not using HTML5

jQuery uses CSS syntax to select elements.

$("#test") - selects by id
$(".test") - selects by class
$("*") - selects all elements
